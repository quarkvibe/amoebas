## ğŸš€ Quick Start

### Install via NPM

```bash
# Install globally
npm install -g @quarkvibe/amoeba

# Initialize Amoeba
amoeba init

# Start the server
amoeba start
```

Visit **http://localhost:5000**

**That's it!** Free tier active immediately. Add your OpenAI key and start generating.

### Install from Source

```bash
git clone https://github.com/quarkvibe/ameoba_client.git
cd ameoba_client
npm install
npm run dev
```
